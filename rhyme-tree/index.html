App overview

ライムの木

韻を集め、育て、ひらめきを積み上げる。検索、図鑑、クイズ、木の成長が、ことばの冒険をひとつにつなぐアプリです。

何ができるか

思いついた語句や読み仮名、母音パターンを集めて、自分だけの韻の図鑑を育てられます。

クイズや木の成長演出で、ことば遊びを続けたくなる流れまで含めて設計しています。

Account

Apple / Google 連携は任意。連携時はクラウドバックアップと復元に対応します。

Growth loop

ログインボーナス、広告ボーナス、容量追加で、集める楽しさを積み上げます。

Main features

  • 語句、読み仮名、母音パターンの登録、検索、コレクション管理
  • クイズと木の成長演出による学習と継続支援
  • Apple / Google アカウント連携とクラウドバックアップ、復元
  • ページ容量追加、ログインボーナス、動画広告ボーナス
